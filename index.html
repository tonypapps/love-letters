About the Project

The content for this project is a creative approach to a standard operating procedures manual for Bash, Git, and GitHub in the form of love letters. The project expresses the basics of these programs and then provides some more advance techniques. Two of these pages express my undying love for the program and one letter was sadly a break-up letter. Each page utalizes a different css design palete. Git commit and push to github were used obsessively.
